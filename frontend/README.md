# Boxmox Frontend

This is the frontend for Boxmox, a personal file hosting server. It is built with React and Vite.

## Theme

- Clean, modern dark interface designed for simplicity and focus, using soft contrasts that stay easy on the eyes.  
- The aesthetic blends deep, muted backgrounds with subtle accent colors that gently guide attention without overwhelming the user.  
- Its minimalist theme reflects efficiency, clarity, and an infrastructure-inspired style that feels both lightweight and professional.
